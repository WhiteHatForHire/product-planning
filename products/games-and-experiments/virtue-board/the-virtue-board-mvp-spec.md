2. One-sentence concept

The Virtue Board is a timed Greek philosophy clue-board game where players select point-value cards, read AI-generated clues about Greek virtues and ethical concepts, type the correct answer, earn points, extend the clock, and receive a post-game learning scorecard showing which concepts they know and which need review.

3. Core design thesis

The Virtue Board is not flashcards.

It is a Greek ethics arcade cabinet.

The goal is to make philosophical vocabulary feel alive, playable, and repeatable without turning it into dry memorization.

The player is training recognition, recall, distinction, and application.

They are not merely matching definitions. They are learning to identify concepts through:

definitions

examples

distorted forms

mythic situations

everyday scenarios

philosopher-style paraphrases

healthy vs unhealthy expressions

practical decision contexts

The AI wedge is not that the AI decides what is true.

The AI wedge is that the AI generates fresh clue variations from a fixed concept bank, so the player cannot simply memorize the card wording.

Core rule:

The game owns the truth. AI writes the clue.

4. Product pillars

Pillar 1: Deterministic concept bank

The game has a fixed canonical concept bank.

Each concept has:

primary answer

accepted aliases

Greek spelling if desired

short definition

expanded definition

healthy expression

distorted lack

distorted excess

related concepts

commonly confused concepts

example scenarios

difficulty level

AI may generate clues, but it may not invent the answer key.

Pillar 2: Fresh board every run

Each run generates a new board from the concept bank.

The same concept can appear in different ways across runs:

direct definition clue

scenario clue

myth clue

distortion clue

Aristotle-style clue

modern life clue

contrast clue

This keeps the game from becoming stale.

Pillar 3: Timed pressure creates recall

The timer makes it feel like a game, not homework.

Correct answers extend the clock.

The player is trying to earn as much “money” as possible before time runs out.

Pillar 4: Wrong answers become learning moments

When the player misses a clue, the game reveals:

correct answer

short explanation

why the clue pointed there

optional distinction from the player’s wrong answer

The point is not punishment. The point is training.

Pillar 5: Scorecard, not just score

The game over screen should become a learning mirror.

It should show:

total score

accuracy

strongest concepts

missed concepts

slow concepts

confused pairs

suggested review set

This is what makes it more than a trivia toy.

7. Core gameplay loop

Run start

Player opens game.

Title screen shows The Virtue Board.

Player clicks Start Trial.

Game selects 25 concept-answer slots from the concept bank.

Game assigns each slot to a category and point value.

AI generates fresh clues for those exact answer slots.

Board appears.

Timer starts.

Card loop

Player selects a card.

Card opens into clue view.

Timer continues running, unless we choose to pause during clue reading. Recommendation: timer continues.

Player types an answer.

Player submits with Enter or button.

Game normalizes answer and checks against accepted aliases.

If correct:

add point value

add time bonus

mark card complete

show brief correct feedback

If incorrect:

reveal correct answer

show short explanation

mark card complete

no points

optional time penalty later, but not v0

Player returns to board.

Loop continues until timer reaches zero or board is cleared.

Run end

Game ends when:

timer reaches zero, or

board is fully cleared

Then show:

final score

cards answered

accuracy

strongest concepts

missed concepts

review recommendations

local leaderboard update

8. Board structure

Board format

Default v0 board:

5 columns

5 rows

25 cards total

Each column is a category.

Each row is a point value.

Rows:

100

200

300

400

500

The higher the value, the more indirect or applied the clue should be.

Category types

A run should select 5 category types from a pool.

Recommended v0 category pool:

1. Definitions

Straight concept recognition.

Example clue:

“This concept refers to practical wisdom: knowing what to do in a specific situation, not merely knowing abstract truths.”

Answer: Phronesis

2. Right Measure

Clues about moderation, restraint, excess, balance, timing.

Example:

“This virtue governs appetite and impulse by finding the right amount, at the right time, in the right way.”

Answer: Sophrosyne

3. Distortions

Identify the concept from its unhealthy form.

Example:

“When this capacity collapses, emotion overwhelms judgment. When it is denied entirely, the person becomes numb and cut off from human feeling.”

Answer: Pathos

4. Healthy vs Distorted

Clues present both the healthy and unhealthy expression.

Example:

“Healthy form: reverence before what is noble. Distorted form: toxic shame or fear of being seen.”

Answer: Aidōs

5. Modern Life

Applied present-day scenarios.

Example:

“You want to post an angry reply, but you pause, choose the right timing, and respond without feeding the fire.”

Answer: Sophrosyne

6. Mythic Lens

Uses Greek myth, gods, heroes, or archetypes.

Example:

“Athena often represents this kind of clear strategic intelligence rather than raw appetite or brute force.”

Answer: Nous or Phronesis, depending on the assigned concept.

Important: Myth clues must be generated carefully so only one answer is clearly intended.

7. Aristotle Mode

Aristotelian practical ethics clues.

Example:

“This is not theoretical wisdom but the judgment that guides action in contingent human affairs.”

Answer: Phronesis

8. Word Roots

Greek meaning and etymological feel.

Example:

“This term points toward truth as unconcealment, not just factual correctness.”

Answer: Aletheia

9. The Polis

Civic, relational, and social-order concepts.

Example:

“This concept names the customs, law, and shared order that shape a community.”

Answer: Nomos

10. Inner State

Concepts about tranquility, emotion, detachment, harmony, or inner steadiness.

Example:

“This state refers to freedom from disturbance, a calmness not easily shaken by external events.”

Answer: Ataraxia

MVP recommendation

For the first build, use these 5 categories:

Definitions

Modern Life

Distortions

Aristotle Mode

Mythic Lens

This gives variety without overcomplicating the clue generator.

10. Answer format

User input

Player types into answer box.

Accepted examples:

sophrosyne

what is sophrosyne

what's sophrosyne

sōphrosynē

temperance, if English aliases are enabled

Normalization

Before checking, normalize:

lowercase

trim whitespace

remove punctuation

remove leading “what is”

remove leading “who is”

normalize diacritics

normalize common transliteration variants

optional: singular/plural normalization

Examples:

What is Sōphrosynē? → sophrosyne

what's phronesis → phronesis

Aletheia! → aletheia

Accepted aliases

Each concept has aliases.

Example:

Sophrosyne

Accepted:

sophrosyne

sōphrosynē

σωφροσύνη

temperance

moderation

self-restraint

But be careful with broad English aliases.

“Moderation” can mean sophrosyne, but if a clue asks for “right measure,” it may be acceptable.

For stricter learning, the game should prefer Greek terms but optionally accept English equivalents.

Recommended answer modes

Add a setting later:

Greek terms required

English aliases accepted

Lenient learning mode

For v0:

Use lenient matching, but always reveal the canonical Greek term.

Example:

Player answers: temperance

Feedback:

“Correct. The Greek term is Sophrosyne.”

11. Clue design

Clue types

Each clue card should have:

assigned answer concept

category

difficulty

clue text

accepted answers

explanation

optional hint

Difficulty logic

100 point clues

Direct definition.

Example:

“This concept means excellence or virtue: the fulfilled quality of a thing or person at their best.”

Answer: Arete

200 point clues

Definition with slight indirection.

Example:

“This is not just a goal, but the end or purpose toward which a life, action, or craft is directed.”

Answer: Telos

300 point clues

Applied scenario.

Example:

“You know what the rule says, but this situation requires judgment about timing, people, and consequences.”

Answer: Phronesis

400 point clues

Distinction between close concepts.

Example:

“This is theoretical wisdom, not practical judgment. It concerns deep truth rather than immediate action.”

Answer: Sophia

500 point clues

Mythic, contrastive, or abstract.

Example:

“A person has power, talent, and status, but forgets limits and insults the order that sustains them.”

Answer: Hubris

Clue constraints

AI-generated clues must:

be answerable

point clearly to one concept

avoid ambiguity with close concepts

be concise

not include the answer in the clue

not use fake direct quotes

not invent philosopher citations

not require obscure scholarship

not depend on facts outside the concept bank unless category requires it

Explanation

Every card should include a short explanation after answer reveal.

Example:

“Sophrosyne is the virtue of right measure, moderation, and restraint. It is not repression. It is appetite and impulse governed by judgment.”

This is essential for learning.

12. Quotes and philosopher references

Important guardrail

Do not let the AI invent exact quotes from Aristotle, Plato, Socrates, or anyone else.

For v0, use:

“In an Aristotelian framing…”

“A Socratic clue might ask…”

“In a Platonic mood…”

“This resembles the question of…”

But avoid quotation marks unless the quote comes from a curated quote bank.

Bad:

“Aristotle said, ‘The right measure is the soul’s crown.’”

Good:

“In an Aristotelian framing, this virtue concerns finding the mean rather than falling into excess or deficiency.”

Later, create a curated quote bank with source references.

Do not show all concepts at once.

Board generation should:

avoid putting highly confusable terms in ambiguous clue categories at high difficulty until the clue generator is reliable

Concept entry format

Each concept should have a structured entry.

Example:

Sophrosyne

Canonical answer: Sophrosyne

Aliases: sophrosyne, sōphrosynē, σωφροσύνη, temperance, moderation, self-restraint

Short definition: restraint, moderation, right measure

Expanded definition: the capacity to govern appetite, impulse, and desire according to reason, timing, and proportion

Healthy expression: choosing the right amount at the right time

Deficiency: impulsivity, indulgence, lack of restraint

Excess/distortion: repression, rigidity, lifeless self-denial

Related: Phronesis, Harmonia, Arete

Confused with: Apatheia, Ataraxia

Example: pausing before acting from appetite or anger

14. AI generation architecture

Generate at beginning of run

Clues should be generated before gameplay begins.

Reasons:

no waiting after card click

smoother gameplay

easier validation

predictable board

cleaner fallback handling

less risk during timed play

Generation flow

Load concept bank.

Select categories.

Select concept-card assignments.

Send structured request to AI.

Receive JSON board.

Validate each clue.

Repair or replace invalid clues.

Start game.

AI should generate

For each card:

clue

short explanation

optional hint

clue style

difficulty

warning if ambiguity exists, optional

AI should not generate

canonical answer

accepted aliases

score value

correctness decision

concept bank truth

leaderboard logic

timer logic

Fallback clues

Every concept should have at least one deterministic fallback clue.

If AI fails:

use fallback clue templates

game still starts

show “standard board” if needed

This is important. The game should not depend entirely on AI availability.

15. Validation layer

Generated clues should be validated before use.

Validation checks

Each clue must:

have a non-empty clue

have a non-empty explanation

not include the answer word directly

not include accepted aliases directly

match assigned category

be under max length

not claim to quote a philosopher directly

not include unsupported citations

not reference concepts outside the bank as required knowledge unless allowed

not point obviously to a different concept

Some checks can be automatic. Others can be simple guardrails.

If validation fails

Options:

regenerate that card once

replace with deterministic fallback

downgrade category to Definitions

Do not block the whole game because one clue failed.

24. Answer validation

Exact matching

First check normalized answer against aliases.

Fuzzy matching

Use fuzzy matching only after exact check.

Useful for:

misspellings

missing diacritics

transliteration variants

Examples to accept:

phronisis → phronesis

sophrosine → sophrosyne

eudaimonia spelled without accents

But do not accept answers that are close to another valid concept.

Example:

Sophia should not be accepted for Sophrosyne.

Logos should not be accepted for Nomos.

Ataraxia should not be accepted for Apatheia.

Wrong answer tracking

If wrong answer matches another concept, record confusion pair.

Example:

Correct: Phronesis
 Player answer: Sophia

Record:

confused Phronesis with Sophia

This powers the scorecard.

25. Content safety / quality

The game is educational and low-risk, but the main quality risk is misinformation.

Guardrails:

no fake quotes

no invented citations

explanations should be concise and conservative

use “Aristotelian framing” instead of pretending direct quote

distinguish classical meanings from Marcus-specific expansions if needed

if a concept is more interpretive, phrase it as “in this game’s framework”

Example:

“In this game’s framework, Aidōs is treated as reverence, appropriate shame, and moral sensitivity before what is noble.”

This avoids overclaiming.
